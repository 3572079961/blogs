# 卡常小寄巧

### I

多维数组连续访问的那一维放在后面有奇效

如倍增预处理

```
int p1[1000005][22];
for(int j=1;(1<<j)<=n;j++){
	for(int i=1;i+(1<<j)-1<=n;i++){
		p1[i][j]=max(p1[i][j-1],p1[i+(1<<j-1)][j-1]);
	}
}

int p2[22][1000005];
for(int j=1;(1<<j)<=n;j++){
	for(int i=1;i+(1<<j)-1<=n;i++){
		p2[j][i]=max(p2[j-1][i],p2[j-1][i+(1<<j-1)]);
	}
}
```

实测在 30 组数据，每组 n=1e6 的情况下， p2 比 p1 快 30s！！！！
